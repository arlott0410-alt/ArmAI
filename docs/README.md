# เอกสาร ArmAI

**[ARMAI-DOCS.md](./ARMAI-DOCS.md)** — เอกสารรวมระบบ ArmAI ฉบับเดียว (single source of truth)

เนื้อหารวมทุกหัวข้อและอัปเดตให้ตรงกับระบบปัจจุบัน:

- สถาปัตยกรรม, โมเดลข้อมูล, Auth, API Routes
- Merchant workspace (Dashboard, Orders, Products, Categories, Knowledge, Promotions, Payment accounts, Telegram, Customers, Channels, Settings)
- Super Admin (Merchants รวม default country/currency, Billing, Support, Audit)
- Billing, Commerce extension, Conversation router, COD, Payment method switching
- Bank account scoping, Parser versioning, Dashboard summary strategy
- Performance, Raw event retention, Telegram operations, BCEL One multi-account
- Scaling (~100 merchants), Enterprise Laos (currency LAK/THB, phone normalization, language)
- Design system (theme, spacing, typography, components)
- การ Deploy, คู่มือติดตั้ง, การทดสอบ, Checklist ความสอดคล้อง
